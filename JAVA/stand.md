# Java 基础语法

## 1. 变量

变量是程序中用于存储数据的占位符，它可以用来保存各种数据类型的值。

变量的命名规则：

1. 变量名只能包含字母、数字、下划线（\_）、美元符号（$）
2. 变量名不能以数字开头
3. 变量名不能是关键字

## 2. 数据类型

Java 语言支持以下数据类型：

- 整型：byte、short、int、long
- 浮点型：float、double
- 字符型：char
- 布尔型：boolean
- 字符串型：String

## 3. 运算符

Java 语言支持以下运算符：

- 算术运算符：+、-、\*、/、%、++、--
- 赋值运算符：=、+=、-=、\*=、/=、%=
- 关系运算符：==、!=、>、<、>=、<=
- 逻辑运算符：&&、||、!
- 位运算符：&、|、^、~、<<、>>、>>>
- 条件运算符：?:

<summary>更多运算符</summary>

```java
// 三元运算符
int a = 10 > 5? 10 : 5;

// instanceof 运算符
```

Java 语言支持以下控制语句：

- if-else 语句
- switch-case 语句
- for 循环
- while 循环
- do-while 循环
- break 语句
- continue 语句
- return 语句
